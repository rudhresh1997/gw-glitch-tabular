Data Availability Statement

The code, configuration files, summary results, and figure-generation scripts used to reproduce the results of this work are publicly available in the project GitHub repository:

https://github.com/rudhresh1997/glitch-tabular-prd

An archived release with a permanent DOI will be provided via Zenodo.

The repository includes model implementations, hyperparameter-tuning workflows, evaluation scripts, interpretability analyses, and scripts used to generate the main figures in the paper. Local cleaned datasets used in this work are documented in the repository structure and data notes.
