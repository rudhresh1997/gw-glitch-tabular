Data Availability Statement

The code, configuration files, summary results, and figure-generation scripts used to reproduce the results of this work are publicly available in the project GitHub repository:

https://github.com/rudhresh1997/glitch-tabular-prd

The dataset used in this work is publicly available from Zenodo:
https://doi.org/10.5281/zenodo.5649212

An archived version of the repository will be provided via Zenodo with a permanent DOI.

The repository includes model implementations, hyperparameter-tuning workflows, evaluation scripts, interpretability analyses, and scripts used to generate the main figures in the paper.
